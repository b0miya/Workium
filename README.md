# Workium — 부서업무관리 시스템

부서원의 월별 업무를 온라인으로 등록·공유·관리하는 웹앱입니다.
별도 설치 없이 URL 하나로 부서원 전체가 즉시 사용할 수 있습니다.

**배포 주소:** https://b0miya.github.io/Workium/

---

## 기술 스택

| 항목 | 내용 |
|---|---|
| 프론트엔드 | HTML / CSS / 바닐라 JavaScript |
| 데이터베이스 | Firebase Firestore (실시간 클라우드 DB) |
| 배포 | GitHub Pages |

---

## 주요 기능

### 전체 보기
- 부서원 전체의 이번 달 업무를 카드 형태로 한눈에 확인
- 월 이동 버튼으로 다른 달 조회 가능

### 개인 업무 관리
- 사이드바에서 부서원 선택 후 연간 업무 조회
- 월별 섹션에서 업무 추가 / 수정 / 삭제

### 공지 및 필수 사이트
- 부서 공지 등록 및 편집
- 자주 사용하는 사이트 링크 등록

### 권한 체계
- **부장:** 부서원 관리, 공지 편집, 필수 사이트 관리
- **일반 부서원:** 전체 보기, 본인 업무 입력

---

## 화면 구성

```
├── 사이드바
│   ├── 전체 보기
│   ├── 부서원 목록
│   ├── 현재 접속자 표시
│   └── 부서원 관리 (부장 전용)
│
└── 메인
    ├── 전체 보기 — 부서원 업무 카드 그리드
    ├── 개인 뷰  — 12개월 업무 테이블
    └── 관리 뷰  — 부서원 추가 / 수정 / 삭제 (부장 전용)
```

---

## 사용 방법

1. 배포 주소 접속
2. 사이드바 하단에서 본인 이름 선택
3. 본인 탭에서 업무 등록

> 부서원 추가 및 초기 설정은 부장 계정으로 로그인 후 진행합니다.

## 로컬 개발 환경 설정

